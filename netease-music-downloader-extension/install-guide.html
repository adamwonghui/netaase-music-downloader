Chrome浏览器插件安装指南

📋 第一步：准备文件

首先需要生成插件图标文件：

  1. 在浏览器中打开 create-simple-icons.html
  2. 页面会自动生成图标下载链接
  3. 依次下载所有图标文件到 icons/ 目录

🎨 生成图标文件

🔧 第二步：安装插件

  1. 打开Chrome浏览器
  2. 在地址栏输入以下地址：
chrome://extensions/
  1. 开启右上角的"开发者模式"开关
  2. 点击"加载已解压的扩展程序"按钮
  3. 选择 netease-music-downloader-extension 文件夹
  4. 确认插件出现在扩展程序列表中
✅ 安装成功后，你应该能在Chrome工具栏看到插件图标

🎵 测试歌曲 1 🎵 测试歌曲 2 📋 完整测试页面
测试检查项：
  • 页面上是否出现红色的"下载MP3"按钮
  • 点击下载按钮是否开始下载
  • 插件图标点击是否弹出正确界面
  • 手动输入链接下载是否正常

📱 使用方法

方法一：页面直接下载（推荐）

  1. 打开网易云音乐网站
  2. 进入任意歌曲页面
  3. 点击页面上的红色"下载MP3"按钮

方法二：插件弹窗下载

  1. 在歌曲页面点击插件图标
  2. 在弹窗中点击"下载当前歌曲"
  3. 或在"手动下载"区域粘贴链接

🐛 常见问题

下载按钮没有出现？

  • 确保在歌曲页面（URL包含 song?id=）
  • 刷新页面重试
  • 检查插件是否启用

下载失败？

  • 某些歌曲可能受版权保护
  • 检查网络连接
  • 确认浏览器下载权限
